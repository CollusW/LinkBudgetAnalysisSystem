# 通信链路仿真与天线约束分析系统 V5.1 工程化接口整理

## 一、TOP 层链路结构

TOP 层不包含核心逻辑，只调用模块：

```text
linkInitConfig
linkLoadOneScenario
linkSimOneScenario
linkAnalyzeOneScenario
linkOutputOneScenario
```

## 二、核心坐标原则

核心链路仿真统一采用 local XYZ 坐标：

- X：前方
- Y：右侧
- Z：向上
- Tx/Rx 坐标单位：m

距离只允许由坐标计算：

```matlab
v_tx_to_rx = rx_xyz - tx_xyz;
d = norm(v_tx_to_rx);
```

不允许将 `d`、`distance`、`range` 作为输入字段。

## 三、接口定义

### 1）linkInitConfig

```matlab
[SimCfg, GeoCfg, RfCfg, AntCfg, AlgoCfg, OutCfg, CfgDiag] = linkInitConfig(simCfgIn);
```

职责：参数初始化。

主要输出：

- `SimCfg`：仿真场景配置
- `GeoCfg`：local XYZ 坐标与姿态配置
- `RfCfg`：射频与链路预算参数
- `AntCfg`：天线模型配置
- `AlgoCfg`：包络/参考链路配置
- `OutCfg`：输出配置

### 2）linkLoadOneScenario

```matlab
[TxNode, RxNode, AntTx, AntRx, InputDiag] = linkLoadOneScenario(SimCfg, GeoCfg, AntCfg);
```

职责：加载 Tx/Rx local XYZ 轨迹与天线模型。

### 3）linkSimOneScenario

```matlab
[LinkResult, SimDiag] = linkSimOneScenario(TxNode, RxNode, AntTx, AntRx, GeoCfg, RfCfg, AntCfg, AlgoCfg);
```

内部调用：

```text
linkPreprocessTrack
linkCalcGeometrySeries
linkCalcAntennaGainSeries
linkCalcBudgetSeries
```

### 4）linkAnalyzeOneScenario

```matlab
[AnalysisResult, AnalysisDiag] = linkAnalyzeOneScenario(LinkResult, RfCfg, AlgoCfg);
```

职责：统计最小 Margin、最大 G_total_required、空间包络等。

### 5）linkOutputOneScenario

```matlab
[OutInfo, OutDiag] = linkOutputOneScenario(LinkResult, AnalysisResult, SimCfg, GeoCfg, RfCfg, AntCfg, OutCfg);
```

职责：输出 CSV/MAT 与图像。

## 五、GPS/WGS84 处理原则

GPS/WGS84 不进入核心链路预算。

若真实输入为经纬高，先用：

```matlab
tools/gps_to_local_xyz/convertGpsCsvToLocalXyz.m
```

转换为：

```text
t,x,y,z,yaw,pitch,roll
```

再进入主仿真。
